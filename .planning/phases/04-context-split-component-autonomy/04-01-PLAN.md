2. **Context menu preventDefault (GridContainer.svelte):** In the `oncontextmenu` handler (around lines 130-139), move `e.preventDefault()` to the FIRST line of the handler, BEFORE the `closest('[data-row][data-col]')` lookup. 3. **FloatingEditor font size (FloatingEditor.svelte):** Add `text-xs` class to the textarea element (around line 148). Grid cells use `text-xs` (12px); the textarea inherits browser default ~16px without it.

These are single-line changes. Do NOT change any context patterns or imports — this task only fixes bugs.
