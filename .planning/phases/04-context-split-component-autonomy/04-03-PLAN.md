<objective>
Migrate all grid components from monolithic getGridContext() to domain-specific context getters, and eliminate all pageActions callback references.

Purpose: This is where component independence is achieved. Each component reads only the contexts it needs and writes directly — no callback threading through props or pageActions. After this plan, deleting any component only removes that feature.

Output: All grid components use domain-specific getters. pageActions pattern completely eliminated from component code. Callback props (onContextMenu, onHeaderClick, onCloseContextMenu) removed from GridContainer.
</objective>
